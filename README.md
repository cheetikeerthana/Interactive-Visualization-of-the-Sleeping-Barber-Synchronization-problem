Sleeping Barber Problem-Interactive Visualization

This project is an interactive visualization of the **Sleeping Barber Problem**,
a classic Operating System synchronization problem.

FEATURES
- Interactive barber shop simulation
- Configurable number of waiting chairs
- Adjustable haircut duration
- Real-time event logs
- Visual representation of barber states (Sleeping / Busy)

OS CONCEPTS USED
- Process Synchronization
- Mutual Exclusion
- Semaphores (Conceptual)
- Resource Sharing
- Deadlock avoidance

TECHNOLOGIES USED
- HTML
- CSS
- JavaScript

HOW TO RUN/EXECUTE
1. Download or clone the repository
2. Open 'index.html' in any modern web browser
3. Click **Add Customer** to start the simulation

LEARNING OUTCOME
This project helps understand how synchronization problems
occur and how they are solved in real operating systems.
